# BoogieBot

## Installation
1. Install Node
2. Install NPM
3. Clone this repo `git clone https://github.com/eriteric/boogiebot.git`
4. Install dependencies: Cd to project root and: `npm install`

## Boogie Down

To start the bot type: `npm run bot`
